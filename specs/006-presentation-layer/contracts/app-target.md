# Contract — App Target & Packaging (US8, D2)

## Checked-in configuration (`App/`)

- `App/project.yml` — XcodeGen spec: one macOS app target `FinanceWorkspace`
  (bundle id `app.openfinance.FinanceWorkspace`), deployment target macOS 15, depends on the
  local SwiftPM package's `FinanceWorkspaceApp` sources via package reference (`path: ..`),
  entitlements + Info.plist wired, DEBUG/RELEASE configs (DEBUG keeps `LocalFolderProvider` via
  the existing `#if DEBUG`).
- Entitlements — defined as `entitlements.properties` **inside `project.yml`** (XcodeGen
  generates `App/FinanceWorkspace.entitlements`, gitignored): iCloud ubiquity container
  `iCloud.app.openfinance.FinanceWorkspace` (matches `AppConfig.iCloudContainerIdentifier`),
  `com.apple.developer.icloud-services: CloudDocuments`, app-sandbox with user-selected
  read-only file access (Finder reveals need no extra entitlement; workspace access flows through
  the container / security-scoped folder pick).
- `App/Info.plist` — **generated by XcodeGen** from `project.yml`'s `info.properties` (app name
  "Finance Workspace", `NSUserActivityTypes: [app.openfinance.navigation]`, minimum system
  version) — gitignored alongside the project.
- `.gitignore` — `App/FinanceWorkspace.xcodeproj/` + `App/Info.plist` (generated, never
  committed).

## CI gate (`.github/workflows/ci-macos.yml`)

New step after `swift test`:

```yaml
- name: Generate app project
  run: brew install xcodegen && xcodegen generate --spec App/project.yml
- name: Build app target (unsigned)
  run: xcodebuild build -project App/FinanceWorkspace.xcodeproj \
         -scheme FinanceWorkspace -destination 'platform=macOS' \
         CODE_SIGNING_ALLOWED=NO
```

Green build of the entitled target = the Milestone-5 packaging gate (SC-009). Signing/notarization
stay developer-machine actions (Phase 7 hardens them).

## Local developer flow

- CLT-only box: `swift build` / `swift run FinanceWorkspaceApp` — full app, DEBUG local-folder
  provider. Never blocked by `App/`.
- Xcode machine: `xcodegen generate --spec App/project.yml && open App/FinanceWorkspace.xcodeproj`
  — run the entitled app against the iCloud container.
